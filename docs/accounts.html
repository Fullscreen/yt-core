--- title: "Yt::Account" h2: "Accounts" ---

Yt::Account represents a YouTube account.

Authentication

All methods of Yt::Account act on behalf of YouTube accounts (e.g.: get the account’s email address).
To use these methods (marked with   below), you need to you need to generate an OAuth client, obtain a refresh token from the account you want to act as, and initialize the object with the refresh token:

{% highlight ruby %} Yt.configuration.client_id = "" ## replace with your client ID Yt.configuration.client_secret = "" ## replace with your client secret account = Yt::Account.new refresh_token: "" ## use the account’s refresh token account.email # => "user@example.com" {% endhighlight %}
