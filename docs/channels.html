--- title: "Yt::Channel" h2: "Channels" ---

Yt::Channel represents a YouTube channel. Initialize using either the YouTube ID:

{% highlight ruby %} channel = Yt::Channel.new id: 'UCwCnUcLcb9-eSrHa_RQGkQQ' # => # {% endhighlight %}

Authentication

Most methods of Yt::Channel retrieve public data from YouTube (e.g.: fetch a channel’s title).
To use these methods (marked with   below), you only need to obtain a Server API key and configure:

{% highlight ruby %} ENV['YT_API_KEY'] = "" ## use your API key channel = Yt::Channel.new id: 'UCwCnUcLcb9-eSrHa_RQGkQQ' ## use any channel ID channel.title # => "Yt Test" {% endhighlight %}

Other methods acts on behalf of YouTube accounts (e.g.: access the analytics of a channel).
To use these methods (marked with   below), you need to obtain an API Client ID/Secret, obtain an access or refresh token from the account you want to act as, and pass the account as the :auth parameter:

{% highlight ruby %} ENV['YT_CLIENT_ID'] = "" ## replace with your client ID ENV['YT_CLIENT_SECRET'] = "" ## replace with your client secret auth = Yt::Account.new refresh_token: "" channel = Yt::Channel.new id: "", auth: auth channel.views # => {total: 345} (the total views of your channel) {% endhighlight %}

Lastly, some methods acts on behalf of YouTube content owners (e.g.: fetch the earnings of a channel).
To use these methods (marked with   below), you need to get an API Client ID/Secret from Google, obtain an access or refresh token from the content owner, and pass the content owner as the :auth parameter:

{% highlight ruby %} ENV['YT_CLIENT_ID'] = "" ## replace with your client ID ENV['YT_CLIENT_SECRET'] = "" ## replace with your client secret auth = Yt::ContentOwner.new id: "", refresh_token: "" channel = Yt::Channel.new id: "", auth: auth channel.ad_impressions # => {total: 123} (the total ad impressions of your channel) {% endhighlight %}
