--- title: "Yt::Models::Channel" h2: "Channels" ---

Yt::Models::Channel represents a YouTube channel. Initialize using either the YouTube ID:

{% highlight ruby %} channel = Yt::Models::Channel.new id: 'UCxO1tY8h1AhOz0T4ENwmpow' # => # {% endhighlight %}

Authentication

Yt::Models::Channel retrieves public data from YouTube (e.g.: fetch a channel’s title and videos).
You only need to get a Server API key from Google and configure:

{% highlight ruby %} ENV['YT_API_KEY'] = "" ## use your API key channel = Yt::Models::Channel.new id: 'UCxO1tY8h1AhOz0T4ENwmpow' ## use any channel ID channel.title # => "Fullscreen" {% endhighlight %}
