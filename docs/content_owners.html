--- title: "Yt::ContentOwner" h2: "Content Owners" ---

Yt::ContentOwner represents a YouTube content owner (an entity that administers content on YouTube).

Authentication

To use any of the methods of Yt::ContentOwner, you need to get an API Client ID/Secret from Google, obtain a refresh token from the content owner, and initialize the object using its unique ID and refresh token:

{% highlight ruby %} ENV['YT_CLIENT_ID'] = "" ## replace with your client ID ENV['YT_CLIENT_SECRET'] = "" ## replace with your client secret content_owner = Yt::ContentOwner.new id: "", refresh_token: "" content_owner.display_name # => "Your CMS" (the name of your CMS) {% endhighlight %}
