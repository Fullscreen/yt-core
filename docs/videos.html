--- title: "Yt::Video" h2: "Videos" ---

Yt::Video represents a YouTube video. Initialize using its YouTube ID:

{% highlight ruby %} video = Yt::Video.new id: 'gknzFj_0vvY' # => # {% endhighlight %}

Authentication

Most methods of Yt::Video retrieve public data from YouTube (e.g.: fetch a video’s title).
To use these methods (marked with   below), you only need to obtain a Server API key and configure:

{% highlight ruby %} Yt.configuration.api_key = "" ## use your API key video = Yt::Video.new id: 'gknzFj_0vvY' ## use any video ID video.title # => "A public video" {% endhighlight %}
