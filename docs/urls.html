--- title: "Yt::URL" h2: "URLs" ---

Yt::URL is a convenience class to identify the kind of a YouTube resource based on its URL: 

{% include doc.html instance="URL#initialize" %}{% include example.html object='url = Yt::URL' method='new' params=' "youtu.be/gknzFj_0vvY"' %}
{% include doc.html instance="URL#kind" %}{% include example.html object='url' method='kind' result=':video' %}
{% include doc.html instance="URL#id" %}{% include example.html object='url' method='id' result='"gknzFj_0vvY"' %}
{% include doc.html instance="URL#canonical" %}{% include example.html object='url' method='canonical' result='"https://www.youtube.com/watch?v=gknzFj_0vvY"' %}
{% include doc.html instance="URL#initialize" %}{% include example.html object='url = Yt::URL' method='new' params=' "https://www.youtube.com/Fullscreen"' %}
{% include doc.html instance="URL#kind" %}{% include example.html object='url' method='kind' result=':channel' %}
{% include doc.html instance="URL#username" %}{% include example.html object='url' method='username' result='"Fullscreen"' %}
{% include doc.html instance="URL#canonical" %}{% include example.html object='url' method='canonical' result='"https://www.youtube.com/user/Fullscreen"' %}

Authentication

Using Yt::URL does not require authentication since it is based on matching a set of hard-coded patterns:

{% highlight ruby %} Yt::URL::PLAYLIST_PATTERNS # => [ # "^(?:https?://)?(?:www.)?youtube\\.com/playlist\\?list=(?[a-zA-Z0-9_-]+)"] # ] Yt::URL::VIDEO_PATTERNS # => [ # "^(?:https?://)?(?:www.)?youtube\\.com/watch\\?v=(?[a-zA-Z0-9_-]+)", # "^(?:https?://)?(?:www.)?youtu\\.be/(?[a-zA-Z0-9_-]+)(?:|/)", # "^(?:https?://)?(?:www.)?youtube\\.com/embed/(?[a-zA-Z0-9_-]+)(?:|/)", # "^(?:https?://)?(?:www.)?youtube\\.com/v/(?[a-zA-Z0-9_-]+)(?:|/)" # ] Yt::URL::CHANNEL_PATTERNS # => [ # "^(?:https?://)?(?:www.)?youtube\\.com/channel/(?[a-zA-Z0-9_-]+)(?:|/)", # "^(?:https?://)?(?:www.)?youtube\\.com/user/(?[a-zA-Z0-9_-]+)(?:|/)", # "^(?:https?://)?(?:www.)?youtube\\.com/(?[a-zA-Z0-9_-]+)(?:|/)" # ] {% endhighlight %}
