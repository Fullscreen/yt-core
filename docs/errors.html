--- title: "Yt::Errors" h2: "Errors" ---

Yt::Errors define error classes that are raised when something goes wrong.

{% highlight ruby %} channel = Yt::Models::Channel.new id: 'this-id-does-not-exist' channel.title # => raise Yt::Errors::NoItems {% endhighlight %}
